# VMA App

Mobilní aplikace vytvořená v rámci předmětu VMA.

## Požadavky

- Android SDK
- Flutter SDK

## Instalace

1. Klonujte repozitář
2. Spusťte `flutter pub get`
3. Spusťte aplikaci pomocí `flutter run`

## Funkce

- **Hlavní obrazovka**: Uložení jména uživatele, počítání návštěv
- **Kalkulačka**: Základní matematické operace s historií
- **BMI Kalkulačka**: Výpočet BMI s ukládáním historie měření
- **O aplikaci**: Informace o aplikaci a použitých technologiích

## Technologie

- Flutter
- Dart
- SharedPreferences
- Material Design 3
